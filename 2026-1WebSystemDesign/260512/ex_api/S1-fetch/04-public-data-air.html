S1-04 · 공공데이터 미세먼지 API (학생 배포용 — mock 데이터)

📦 이 데모는 로컬 mock JSON을 사용합니다.

실제 공공데이터포털 API와 같은 구조의 응답을 ./mock/air-quality.json에서 읽어옵니다.

실제 API로 바꾸고 싶다면 아래 USE_REAL을 true로 + API_KEY 입력 → 새로 고침

대기 중

왜 mock으로 시작하나

  • API 키 발급/도메인 등록 안 해도 fetch 흐름을 그대로 익힐 수 있다
  • 실제 API로 전환은 USE_REAL = true 한 줄
  • 응답 구조는 실제 API와 동일하게 만들어 둠 → 코드 그대로 동작
