S2-01 · 첫 지도 띄우기 (라이브 데모 1)

✅ Leaflet + OpenStreetMap — 키 발급, 카드 등록, 도메인 등록, 회원가입 모두 불필요. <script>만 불러오면 끝.

핵심 두 줄 요약

  • L.map(id).setView([lat, lng], zoom) — 지도 생성 + 중심·줌 설정
  • L.tileLayer(URL).addTo(map) — 타일 이미지(OSM)를 지도에 얹기

지도가 안 뜨면?

  • leaflet.css 누락 — 안 불러오면 타일이 깨져서 보임
  • 컨테이너 #map에 height가 없음 — Leaflet은 명시적 높이 필요
  • 인터넷 차단 — CDN(unpkg.com, tile.openstreetmap.org) 접근 가능한지 확인
