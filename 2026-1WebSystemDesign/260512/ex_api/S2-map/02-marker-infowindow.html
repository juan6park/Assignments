S2-02 · 마커 + 팝업

Leaflet에서는 InfoWindow를 Popup이라 부른다. 카카오 InfoWindow와 거의 같은 개념.

비교

// 카카오 (4줄)
const marker = new kakao.maps.Marker({ position: ..., map, title: '...' });
const info = new kakao.maps.InfoWindow({ content: '...' });
kakao.maps.event.addListener(marker, 'click', () => info.open(map, marker));

// Leaflet (1줄)
L.marker([lat, lng]).addTo(map).bindPopup('...');   // 클릭 시 자동으로 열림

학생 자주 묻는 것

  • 좌표 순서: Leaflet도 [위도, 경도] — 위도 먼저 (카카오와 동일)
  • 여러 팝업 동시 열림? — Leaflet은 기본적으로 1개만 열림 (직전 자동 닫힘)
  • 마우스오버 툴팁: .bindTooltip('이름')
