S2-05 · 마커 30개 + 클러스터링

마커가 많을 때 가까운 것끼리 묶어 보여준다. 줌 인하면 풀림.

핵심 3줄

const cluster = L.markerClusterGroup();         // 1) 그룹 만들기
markers.forEach(m => cluster.addLayer(m));       // 2) 마커 넣기
map.addLayer(cluster);                          // 3) 지도에 그룹 통째로

플러그인 leaflet.markercluster는 별도 CDN으로 로드. CSS 2개 + JS 1개.
