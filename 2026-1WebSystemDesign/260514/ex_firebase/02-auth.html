Note12-02 · Google 로그인 (라이브 데모 2)

슬라이드: Auth 핵심 4함수 + Google 로그인 5줄 + onAuthStateChanged 패턴

로그인하지 않은 상태입니다.

코드 핵심

// Google 로그인
signInWithPopup(auth, new GoogleAuthProvider());

// 로그인 상태 *구독* — 변할 때마다 자동 호출
onAuthStateChanged(auth, (user) => {
  if (user) showLoggedIn(user);
  else      showLoggedOut();
});

자주 막히는 곳

  • 팝업 안 뜸 — Auth → Sign-in method 탭에서 Google 활성화 안 함
  • "popup-blocked" — 브라우저 주소창 옆 아이콘으로 팝업 허용
  • 익명 로그인 안 됨 (auth/operation-not-allowed) — Auth → Sign-in method 탭에서 Anonymous 활성화 안 함. 콘솔에서 Enable 누르고 저장
  • 새로고침 후에도 로그인 유지됨 — 정상. onAuthStateChanged가 세션 복원
  • UID — 24자 문자열, 사용자마다 고유. Firestore 문서의 authorId와 연결할 키
