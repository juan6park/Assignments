Note12-04 · 실시간 메모장 ⭐ (라이브 데모 3 — 클라이맥스)

슬라이드: onSnapshot — 실시간 구독 · 두 탭 분할로 즉시 동기화 시연

🔥 이 페이지를 두 개 탭으로 열어 보세요. 한 탭에서 메모를 추가하면 다른 탭에 즉시 나타납니다. 페이지 새로고침 없이. 학기 프로젝트의 클라이맥스 모양.

대기 중

코드 핵심

const q = query(collection(db, 'memos'), orderBy('createdAt', 'desc'));

// 한 번 호출하면 데이터 변할 때마다 자동 콜백
onSnapshot(q, (snapshot) => {
  const memos = snapshot.docs.map(d => ({ id: d.id, ...d.data() }));
  render(memos);
});

두 탭 시연 흐름

  1. 이 페이지를 두 개 탭으로 열기 (Live Server URL 같은 것)
  2. 두 탭에서 모두 로그인 (다른 Google 계정이면 더 좋음)
  3. 한 탭에서 메모 추가 → 다른 탭에 즉시 나타남 (새로고침 X)
  4. 다른 탭에서 메모 추가 → 이 탭에 즉시 나타남
  5. 본인 메모 옆 "삭제" 버튼 → 양쪽 탭에서 동시에 사라짐

자주 막히는 곳

  • permission-denied — 보안 규칙이 read를 막음 → 05 데모에서 풀기
  • onSnapshot 콜백 안 호출됨 — 보안 규칙 read 거부 또는 인증 안 됨
  • 날짜가 "대기"로 보임 — serverTimestamp는 서버 도달 후에 채워짐 (잠시 후 자동 갱신)
